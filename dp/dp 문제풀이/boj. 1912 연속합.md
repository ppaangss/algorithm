# 해결과정

선형 dp + 조건 (연속성) 이 달린 문제였다.

즉, 이 문제에서 dp 조건은 해당 $dp[i]$ 는 해당 숫자 구간 $num[i]$ 를 반드시 포함해야 해야하는 것이 조건이다.

왜냐하면 연속성이라는 조건을 유지하기 위해서이다. 만약 $dp[i]$ 가 `num[i]` 를 포함하지 않아도 된다고 정의하는 순간 연속성을 유지할 수 없다.
