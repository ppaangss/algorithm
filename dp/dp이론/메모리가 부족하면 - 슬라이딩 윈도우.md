```
// N이 10만이더라도 딱 2행만 사용하는 방식
vector<vector<int>> dp(2, vector<int>(3, 0)); 

// i % 2 를 사용해서 0행과 1행을 번갈아가며 사용 (공간 최적화)
for (int i = 2; i <= N; i++) {
    int cur = i % 2;
    int prev = (i - 1) % 2;
    dp[cur][0] = (dp[prev][1] + dp[prev][2]) % 9901;
    // ... 나머지도 동일
}
```

메모리가 부족할 때 진짜 유용한 기술은 '숫자 규칙 찾기'가 아니라 '슬라이딩 윈도우'입니다.

이 방식은 **논리(2차원)는 유지하면서 메모리만 획기적으로 줄이는** 아주 세련된 방법입니다. 억지스러운 1차원 점화식보다 훨씬 가치 있는 공부죠.
