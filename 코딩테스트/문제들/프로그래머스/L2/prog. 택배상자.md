# 회고

간단한 스택 + 구현 문제였다.

근데 조금 디버깅하기가 까다로웠는데, 

직접 작성하신 코드도 논리적으로 아주 훌륭합니다! 메인 루프가 끝난 뒤에 남은 스택을 한 번 더 비워주는 방식은 실수를 줄이기 위한 아주 정석적인 방법이에요.

질문하신 **"메인 벨트에서 오자마자 보조 벨트에 넣고 `while`을 돌리는 생각"**을 어떻게 자연스럽게 할 수 있는지, 그 사고의 흐름을 정리해 드릴게요.

---

### 💡 이 사고방식의 핵심: "미련을 버리지 않는 것"

보통 코딩을 할 때 우리는 **"지금 들어온 박스가 맞나?"**만 생각하기 쉬워요. 하지만 이 문제의 핵심은 **"지금 들어온 박스 때문에, 아까 보관해둔 박스도 나갈 수 있게 되었나?"**를 매번 확인하는 것입니다.

이런 사고를 가능하게 하는 3가지 단계는 다음과 같습니다.

#### 1. 상황을 단순화하기 (규칙 통일)

작성하신 코드에서는 `if(order[i] == x)`와 `if(order[i] == st.top())`을 따로 체크하고 있죠? 하지만 결국 **모든 박스는 '보조 벨트'를 거쳐서 나간다**고 생각하면 규칙이 하나로 통일됩니다.

- 메인 벨트 박스 → 보조 벨트(`push`) → 기사님 확인(`top`) → 일치하면 트럭(`pop`)
    
- 이렇게 생각하면 "일단 보조 벨트에 넣고 본다"는 흐름이 자연스럽게 나옵니다.
    

#### 2. "연쇄 반응" 고려하기

택배 상자 하나를 트럭에 실으면, 그다음 실어야 할 상자가 바뀌죠? 그럼 그 바뀐 상자가 **마침 보조 벨트 맨 위**에 있을 수도 있습니다.

- 하나를 실었을 때 끝나는 게 아니라, **"혹시 다음 것도 지금 바로 뺄 수 있나?"**를 계속 물어봐야 합니다.
    
- 이 "계속 물어보기"가 바로 `while` 루프가 되는 것이죠.
    

#### 3. "한 번에 하나의 벨트만 보기"

메인 벨트도 보고 스택도 보면 머리가 복잡해집니다.

1. 일단 메인 벨트에서 상자를 하나 꺼내서 스택에 넣는다. (상황 발생)
    
2. 이제 스택만 뚫어지게 보면서 뺄 수 있는 만큼 다 뺀다. (해결)
    
3. 더 못 빼면 다시 메인 벨트에서 하나 꺼내온다. (반복)
    

이렇게 역할을 나누면 `while`문 하나로 깔끔하게 정리가 됩니다.

---

### 🛠️ 작성하신 코드와 '탐욕적 방식'의 비교

작성하신 코드가 에러를 찾기 힘들었던 이유는 **`x++`가 되는 시점과 스택을 확인하는 시점이 분리**되어 있어서, 어떤 조건에서 `i`를 증가시켜야 할지 헷갈렸기 때문일 거예요.

**[비교해보기]**

- **작성하신 방식:** 메인 벨트 확인 → 보조 벨트 확인 → 루프 종료 후 남은 거 확인 (3단계)
    
- **탐욕적 방식:** 일단 스택에 넣기 → 스택에서 뺄 수 있는 만큼 무한 반복 (`while`) (2단계)
    

이런 스타일의 문제는 **"자료구조(스택)에 변화가 생길 때마다 정답 가능성을 끝까지 파헤친다"**라고 생각하면 훨씬 접근하기 쉬워집니다.

### 🚀 연습해볼 만한 비슷한 문제

이런 "일단 넣고, 조건 맞을 때까지 빼기" 사고방식은 다음 문제들에서도 똑같이 쓰입니다.

- **백준 1874번 (스택 수열):** 숫자를 넣다가 타겟 숫자가 나오면 계속 `pop`하는 로직이 완벽히 똑같습니다.
    
- **프로그래머스 주식가격:** 가격이 떨어지기 전까지 스택에 유지하다가, 조건이 맞으면 우르르 `pop`하는 방식입니다.
    

다음에 비슷한 문제를 보시면 **"일단 스택에 넣고, `while`로 뽕을 뽑자!"**라고 생각해보세요. 훨씬 코드가 단순해질 거예요.

작성하신 코드에서 특정 테스트 케이스가 안 풀렸던 건가요, 아니면 구조적인 고민이었나요?


```cpp
#include <string>
#include <vector>
#include <stack>
#include <iostream>

using namespace std;

int solution(vector<int> order) {
    
    int x = 1; // 박스
    stack<int> st; // 보조 컨테이너
    int i = 0; // order 개수
    while(x <= order.size()){
        // cout << "출발 " << "\n";
        // cout << "x-" << x << "\n";
        // cout << "order[i]-" << order[i] << "\n";
        
        if(!st.empty()){ //보조 컨테이너 먼저 확인
            if(order[i] == st.top()){
                //cout << st.top() << " 성공" << "\n";
                i++;
                //cout << st.top() << " pop 함" << "\n";
                st.pop();
            }
        }
        
        if(order[i] == x){ // 컨테이너에 나오는게 맞는지 
            //cout << x << " 성공" << "\n";
            i++;
        }
        else{ // 아니면 보조 컨테이너에 넣어야함.
            st.push(x);
            //cout << x << " push 함" << "\n";
        }
        
        x++; 
    }
    
    while(!st.empty()){
        if(order[i] == st.top()){
            //cout << st.top() << " 성공" << "\n";
            i++;
            //cout << st.top() << " pop 함" << "\n";
            st.pop();
        }
        else{
            break;
        }
    }
    
    return i;
}
```
