
#  절댓값, 거듭제곱, 제곱근, 피타고라스

`#include <cmath>` 에서 사용 가능하다.

| 함수            | 기능                            | 예시                      |
| ------------- | ----------------------------- | ----------------------- |
| `abs(x)`      | 절댓값 구하기                       | `abs(-5)` → `5`         |
| `pow(a, b)`   | 거듭제곱 ($a^b$)                  | `pow(2, 10)` → `1024.0` |
| `sqrt(x)`     | 제곱근 ($\sqrt{x}$)              | `sqrt(16)` → `4.0`      |
| `hypot(x, y)` | 피타고라스 정리 ($\sqrt{x^2 + y^2}$) | `hypot(3, 4)` → `5.0`   |

pow와 sqrt는 결과값을 `double` 로 반환한다.

# 올림, 내림, 반올림

`#include <cmath>` 에서 사용 가능하다.

| 함수         | 기능      | 예시                   |
| ---------- | ------- | -------------------- |
| `ceil(x)`  | 올림 (천장) | `ceil(3.1)` → `4.0`  |
| `floor(x)` | 내림 (바닥) | `floor(3.9)` → `3.0` |
| `round(x)` | 반올림     | `round(3.5)` → `4.0` |

`a/b` 를 올림 하고 싶으면 ceil 보다 `(a+b-1) / b` 를 쓰는게 더 빠르고 정확하다.



# max, min

- `max(a, b)` / `min(a, b)`: 두 값 중 큰/작은 값 반환.
- C++11 부터는 `max({a,b,c,d})`  처럼 중괄호를 써서 여러 개 비교가 가능하다.
	- `max({1,2,3,4})` -> `4`

# 나머지 연산

나머지 연산(`%`): 안전하게 양수 나머지를 얻으려면 `(a % b + b) % b` 공식을 활용하는게 좋다.
