요소 반복문

```c++
for (const auto& x : um) {
    sum *= x.second + 1;
}

for (auto x : um) {
    sum *= x.second + 1;
}
```



| 항목  | `auto x`   | `const auto& x` |
| --- | ---------- | --------------- |
| 동작  | 값 복사       | 참조              |
| 메모리 | 추가 복사 발생   | 복사 없음           |
| 성능  | 살짝 느릴 수 있음 | 더 효율적           |
| 안전성 | 수정 가능      | 읽기 전용           |
따라서 읽기만 할 거면 첫번째꺼 쓰면 좋을 것 같다.
