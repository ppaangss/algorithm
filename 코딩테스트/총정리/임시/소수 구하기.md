
# 구현 (소수를 찾는 두 가지 상황)

1. 단일 숫자 판별
2. 특정 범위의 모든 소수 찾기

---
## 1. 단일 숫자 판별

**단일 숫자가 소수인지 빠르게 확인할 때 사용한다.**

어떤 수 N이 소수인지 확인할 때 2부터 N-1까지 모두 나눠볼 필요가 없다.
에라토스테네스의 접근에 따라 판별은 N의 제곱근 까지 확인하면 충분하다.
- 1 X 16 = 16 _________
- 2 X 8 = 16   ___               |
- 4 X 4 = 16         |   대칭   대칭
- 8 X 2 = 16   ___               |
- 16 X 1 = 16 _________

가운데 약수 4를 기준으로 각 등식이 대칭적인 형태를 보인다는 것을 알 수 있다.  
예를 들어 2 X 8 = 16은 8 X 2 = 16과 대칭이다.
그렇기에 특정한 자연수 X가 소수인지 확인하기 위하여 바로 가운데 약수까지만 **나누어 떨어지는지** 확인하면 된다.

다시 말해 **제곱근까지만(가운데 약수까지만) 확인하면 된다!!**
약수는 항상 쌍으로 존재하기 때문이다.

```c++
#include <iostream>
#include <cmath>
using namespace std;

bool isPrime(int n) {
	if (n <= 1) return false; // 1보다 작은 수 제거
	if (n == 2) return true; // 2는 소수다.
	if (n % 2 == 0) return false; // 짝수 제거

	for (int i = 3; i * i <= n; i += 2) {
		if (n % i == 0) return false;
	}
	return true;
}
```

### 시간복잡도

$O(\sqrt{n})$

## 2. 특정 범위의 모든 소수 찾기 -

**대량의 범위 내 소수를 한꺼번에 구할 때 사용한다.**

에라토스테네스의 체 알고리즘을 이용해 구현한다.

### 동작 원리

1. 2부터 N까지의 모든 수를 나열한다.
2. 아직 지워지지 않은 가장 작은 수 p를 찾는다.
3. p의 배수들을 모두 지운다.
4. 더 이상 지울 배수가 없을 때까지 반복한다.

```c++
#include <iostream>
#include <vector>
#include <cmath>

void sieveOfEratosthenes(int n) {
    // 0부터 n까지의 유무를 저장하는 벡터 (true로 초기화)
    std::vector<bool> prime(n + 1, true);
    prime[0] = prime[1] = false;

    for (int p = 2; p * p <= n; p++) {
        // prime[p]가 true라면 p는 소수
        if (prime[p] == true) {
            // p의 배수들을 false로 변경
            for (int i = p * p; i <= n; i += p)
                prime[i] = false;
        }
    }

    // 결과 출력
    for (int p = 2; p <= n; p++) {
        if (prime[p])
            std::cout << p << " ";
    }
}
```

### 시간복잡도

$O(n \log \log n)$
