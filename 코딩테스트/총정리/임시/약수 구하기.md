# 약수를 찾는 두 가지 상황 (소수랑 똑같음)

1. 단일 숫자 판별
2. 특정 범위의 모든 소수 찾기

## 1. 단일 숫자의 약수 구하기

특정 수 $N$의 약수를 하나씩 찾거나 개수를 셀 때 사용합니다. 소수 판별과 마찬가지로 $N$의 제곱근($\sqrt{N}$) 까지만 확인하면 충분합니다.

약수는 항상 쌍으로 존재합니다. ($N=16$일 때, $2 \times 8 = 16$이므로 2를 찾으면 8은 자동으로 알 수 있습니다.)

- $j \times j = i$ 인 경우 (제곱근): 약수 1개 추가
- $j \times j \neq i$ 인 경우 (쌍이 존재): 약수 2개 추가

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

// 약수의 개수만 구할 때
int countDivisors(int n) {
    int cnt = 0;
    for (int i = 1; i * i <= n; i++) {
        if (n % i == 0) {
            if (i * i == n) cnt++;   // 제곱근인 경우 (ex: 4*4=16)
            else cnt += 2;           // 쌍이 있는 경우 (ex: 2*8=16)
        }
    }
    return cnt;
}
```

### 시간복잡도

$O(\sqrt{n})$

## 2. 특정 범위의 모든 약수 개수 찾기

1부터 $E$까지 모든 숫자의 약수 개수를 한꺼번에 구할 때 사용합니다. "배수의 성질"을 이용한 에라토스테네스의 체(Sieve) 알고리즘을 사용합니다.

### 동작 원리

1. 1부터 $N$까지의 개수를 저장할 배열을 0으로 초기화합니다.
2. 1부터 $N$까지 숫자를 하나씩 선택합니다 ($i$).
3. $i$의 배수들을 찾아가서 "너는 $i$를 약수로 가진다"는 의미로 카운트를 1씩 증가시킵니다.
4. 이 방식은 "이 숫자의 약수가 뭐지?"라고 묻는 대신, "이 숫자는 누구의 약수가 될 수 있지?"라고 거꾸로 접근하는 방식입니다.


```cpp
#include <iostream>
#include <vector>

using namespace std;

vector<int> getAllDivisorsCount(int n) {
    // 0으로 초기화된 배열 준비
    vector<int> count(n + 1, 0);

    for (int i = 1; i <= n; i++) {
        // i의 배수들에 대해 모두 +1 (i는 그들의 약수이므로)
        for (int j = i; j <= n; j += i) {
            count[j]++;
        }
    }
    return count;
}
```

시간복잡도

$O(n \log n)$

_(조화급수의 합 원리에 의해 $n(1 + 1/2 + 1/3 + \dots + 1/n) \approx n \ln n$)_

# 소수 vs 약수

| 구분          | 단일 숫자 (N)              | 특정 범위 (1∼E)            |
| --------------- | -------------------------- | -------------------------- |
| 핵심 아이디어     | 제곱근($\sqrt{N}$) 까지만 확인 | 배수(Multiple)**를 찾아가며 카운팅 |
| 소수(Prime)   | $O(\sqrt{N})$              | $O(E \log \log E)$         |
| 약수(Divisor) | $O(\sqrt{N})$              | $O(E \log E)$              |
| 비고          | 낱개로 물어볼 때 유리               | 대량으로 물어볼 때 압도적 유리          |
