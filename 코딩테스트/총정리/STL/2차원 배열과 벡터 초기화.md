# fill 함수

배열의 모든 요소를 0이 아닌 다른 값(예: -1, 99)로 채워야 할 때 가장 안전한 방법이다.

헤더: `<algorithm>`

`fill(시작_주소, 끝_주소, 값);`

```c++
#include <algorithm>

vector<int> arr(10);
std::fill(arr, arr + 10, -1); // 모든 요소를 -1로 초기화
```

# memset 함수

메모리 블록을 직접 바이트 단위로 채우는 방식입니다. `<cstring>` 헤더가 필요합니다.

- 주의: 바이트 단위로 동작하기 때문에 0이나 -1 이외의 정수값으로 초기화할 때는 사용하면 안 됩니다. (예: 1로 초기화하려고 하면 예상치 못한 큰 값이 들어감)
- 용도: 대규모 배열을 빠르게 `0`으로 밀어버릴 때 주로 사용합니다.

```cpp
#include <cstring>

int arr[100];
memset(arr, 0, sizeof(arr)); // 모든 요소를 0으로 초기화
```
