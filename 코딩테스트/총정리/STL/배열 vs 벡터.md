
# 배열과 벡터 무엇을 써야하나

|**구분**|**정적 배열 (int a[])**|**벡터 (std::vector)**|
|---|---|---|
|**권장 상황**|크기 제한이 작고 고정된 경우 (PS의 대부분)|데이터 크기가 가변적이거나 매우 큰 경우|
|**메모리 위치**|데이터 영역 / 스택|힙 (Heap)|
|**초기화**|`memset(arr, 0, sizeof(arr))`|`v.assign(n, 0)`|
|**속도**|**최상** (미세하게 빠름)|**상** (무시할만한 수준이지만 반복문이 많으면 차이 남)|

N이 매우 큰 상황에서는 메모리 초과가 나올 가능성이 있음. 따라서 벡터를 사용하는 것이 바람직하다.

하지만 시뮬레이션 같은 반복횟수가 많은경우 미세하게 정적 배열이 크다. 

따라서 적절하게 잘 사용하면 좋을 것 같다.
