
# 우선순위 큐

`std::priority_queue`를 사용할 때 세 번째 인자는 우선순위(정렬 기준)를 결정하는 비교 클래스이다.

최소힙은 외우자. 많이 쓰지 않을까? 

**최소힙**

```cpp
#include <iostream>
#include <queue>
#include <vector>

using namespace std;

int main() {

	// 기본은 최대 힙
    priority_queue<int> pq;
    // 세 번째 인자에 std::greater<int>를 넣어 최소 힙 구현
    priority_queue<int, vector<int>, greater<int>> pq;
    
```

**사용자 정의 정렬 (Struct/Class)**

```cpp
#include <iostream>
#include <queue>
#include <vector>

using namespace std;

struct Compare {
    bool operator()(int a, int b) {
        return a > b; // greater와 동일하게 동작 (Min Heap)
    }
};

// 선언부
priority_queue<int, vector<int>, Compare> customPq;

```
