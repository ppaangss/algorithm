
# 출력에 소수점을 강제로 포함하기

"소수점 첫째 자리까지 출력하라"거나 "오차 범위를 허용한다" 라는 조건이 문제에 있는 경우

만약에 내가 답이 8.0이다. 그러면 그냥 출력시 8이 나온다. 하지만 답은 8.0을 요구할 경우 오답 처리가 된다. 

**fixed와 precision 사용하기**

- `fixed`: 소수점을 고정시키겠다는 뜻입니다. 즉, "지수 형태(`8e+00`)로 쓰지 말고 무조건 소수점 형태로 써라!"라고 명령하는 거죠.
- `precision(n)`: 소수점 아래 `n`번째 자리까지 무조건 표시하라는 뜻입니다. 나머지는 반올림합니다.

만약에 "절대/상대 오차는 $10^-9$까지 허용한다." 라고 한다면 precision(9)를 해서 9번째 자리 수 까지는 같으면 된다. 

```cpp
cout << fixed;
cout.precision(9);
cout << maxNum << "\n"
```

또는 기존 printf 문을 이용해 출력하는 방법도 있다.

```cpp
printf("%.1f\n", maxNum);
```
