# 알고리즘 큰 틀


그리디 -> 완전탐색 -> DP, 이진탐색, ...

완전탐색이 안되는 경우는 시간복잡도의 문제일 수 밖에 없어.

그러면 탐색이 아닌 시간복잡도를 줄일 수 있는 알고리즘을 써야하는거.

그게 이제 DP라던가, 이진탐색이라던가 될  수 있다.


1. 간단한 그리디를 생각한다.
2. 완전탐색을 생각한다.
	1. 브루트포스 (그리디랑 비슷함)
	2. BFS
	3. 재귀 (DFS, 백트래킹)
3. 효율적인 탐색
	1. DP
	2. 이진탐색
