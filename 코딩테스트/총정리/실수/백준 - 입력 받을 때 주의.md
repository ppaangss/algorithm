
# 입력 데이터가 공백이 없을 때 주의

백준 1051을 풀 때 였다. 숫자를 넣는데 벡터에 들어가지가 않는 문제가 생김.

예시로 12345가 입력으로 주어지고 이것을 벡터에 넣으려고 할 때 주의해야한다.

왜냐하면 공백이 없기 때문에 그냥 넣을려고 하면 안들어가진다.


ex) 입력이 12345 이렇게 들어올 때

이렇게 하면 벡터에 숫자가 안들어가진다.

```cpp
vector<vector<int>> v(N,vector<int>(M,0));

for (int i = 0; i < N; i++) {  
	for (int j = 0; j < M; j++) {
		cin >> v[i][j];
	}
}
```


해결은 string으로 받아 벡터에 넣는 작업을 해주면 된다.

```cpp
vector<vector<int>> v(N,vector<int>(M,0));

for (int i = 0; i < N; i++) {  
	string tmp; cin >> tmp;
	for (int j = 0; j < M; j++) {
		v[i][j] = tmp[j] - '0';
	}
}
```
