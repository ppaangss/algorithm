# 입력 한 줄 씩 받기

`cin >> 변수` 는 기보적으로 공백, 탭, 엔터를 만나면 입력이 끝났다고 판단한다.

따라서 한 줄 입력받는 것을 원할 때는 `getline(cin, 변수)` 를 써야한다.


```cpp
#inclue <iostream>

int main(){
	string s;
	getline(cin, s);
}

```

버퍼에 남은 엔터를 읽고 바로 종료될 수 있다. 그럴 때를 대비해 `cin.ignore();` 를 해줘 남은 엔터를 제거해줘야 한다.

```cpp
#inclue <iostream>

int main(){
	
	int n;
	string s;
	
	cin >> n
	cin.ignore();
	getline(cin, s); // ignore 없으면 엔터가 입려됨.
}
```


# 특정 문자까지만 읽는 법

특정 문자까지만 입력을 받고 싶을 수 있다. 그럴 때는 `getline(입력스트림, 저장할변수, 구분자);` 를 사용하면 된다.

이때, 구분자를 만나면 그 구분자는 변수에 저장되지 않고 버퍼에서 지워버린다.

```cpp
#include <iostream>
#include <string>
#include <sstream> // 문자열을 자를 때 유용함

using namespace std;

int main() {
    string input = "Hong,25,Student";
    stringstream ss(input); // 문자열을 '입력 스트림'처럼 취급함
    
    string name, age, job;

    getline(ss, name, ','); // 'Hong' 읽고 멈춤
    getline(ss, age, ',');  // '25' 읽고 멈춤
    getline(ss, job);       // 나머지 'Student' 읽음 (기본값은 \n)

    cout << "Name: " << name << "\nAge: " << age << "\nJob: " << job << endl;

    return 0;
}
```
