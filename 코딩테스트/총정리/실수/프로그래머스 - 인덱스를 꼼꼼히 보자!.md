
# 인덱스를 꼼꼼히 보자.

백준 문제를 풀고 있었지만 디버깅 없이 순수 출력만으로 문제를 해결하는 연습중이었다.

이제 백준 1051 문제를 푸는데 이제 2차원 배열을 순회하는 브루트포스 문제였다.

이제 인덱스 조절이 중요한데 디버깅이 안될 때 인덱스를 벗어나는 문제가 나오면 찾기가 굉장히 어렵다. 그래서 인덱스 관련 된 건 무조건 꼼꼼히 보고 가자!!!


```cpp

int Size = 1;
bool flag = true;

for (int i = 0; i <= N - Size - 1; i++) {
    for (int j = 0; j <= M - Size - 1; j++) {
        // 맨 왼쪽이 어디냐
        //cout << "i-" << i << "_j-" << j << "\n";
        int num = v[i][j];

        if (num == v[i + Size][j] && num == v[i + Size][j + Size] && num == v[i][j + Size]) {
            flag = true;
        }
        // 숫자 확인
        //cout << v[i][j] << v[i + Size][j] << v[i + Size][j + Size] << v[i][j + Size] << "\n";
        //cout << "flag-" << flag << "\n";
        
    }
}

```

이런 순회를 풀고 있을 때 인덱스가 맞는지 부터 꼼꼼히 출력문으로 확인해보자.

주석에 있는 `cout << "i-" << i << "_j-" << j << "\n"` 를 통해 인덱스가 맞는지 확인할 수 있었다.
