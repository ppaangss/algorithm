
## 개요

algorithm 라이브러리
범위 내에서 연속적으로 중복된 원소를 처리할 때 사용
전체 범위에서 모든 중복을 없애고 싶으면 반드시 정렬을 필수로 해야한다.

## 설명

unique는 중복되지 않은 원소들을 앞으로 모아놓고 뒤에 중복된 쓰레기값들을 모아놓는다.
예를들어 {1,2,2,3,3,3,4} 를 unique 할 시 {1,2,3,4,2,3,3} 으로 변환시켜준다.
2개의 2가 중복되므로 2 1개, 3개의 3이 중복되므로 3 2개가 맨 뒤로 가게된다.

이때, unique의 반환값은 쓰레기 값의 시작점을 반환한다.
따라서 unique의 반환값 부터 end() 까지가 쓰레기 값이 남는 구간이다.

전체 범위에서 모든 중복을 없애고 싶으면 반드시 정렬을 필수로 해야한다.
원리는 인접한 같은 값을 없애주는 원리이기 때문이다.
예를들어 {1,2,2,3,2,4} 를 unique 할 시 {1,2,3,2,4,2} 로 변환시켜준다.
뒤에 2는 중복으로 없애주지만 뒤에 2는 없애주지 않는다.

erase와 함께 사용하면 중복을 없앤 컨테이너를 얻을 수 있다.

시간복잡도는 O(N) 이다.
원소들이 서로 비교 가능해야 한다.

## 코드 예시

```c++
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main() {

    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    cout.tie(NULL);

    vector<int> v = { 1, 2, 2, 3, 3, 3, 4, 2, 1 };
    vector<int> v1 = { 1, 2, 3, 2 };

    // 1. 모든 중복을 제거하기 위해 정렬
    sort(v.begin(), v.end()); // {1, 1, 2, 2, 2, 3, 3, 3, 4}

    // 2. unique 적용
    // 결과: {1, 2, 3, 4, (중복된 나머지 숫자들이 뒤로 모임)}
    auto last = unique(v.begin(), v.end());

    // 3. 실제 물리적 크기 줄이기
    v.erase(last, v.end());

    for (int n : v) std::cout << n << " "; // 출력: 1 2 3 4
}
```
