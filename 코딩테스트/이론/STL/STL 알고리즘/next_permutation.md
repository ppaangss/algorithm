
현재의 수열보다 사전순으로 바로 다음에 오는 수열을 찾는다.

라이브러리 : [[algorithm]]
반환값 : 다음 순열이 존재하면 **true**를 반환, 마지막 순열에 도달 해 더 이상 다음이 없으면 **false**를 반환하고 수열을 다시 첫번째 상태 (오름차순)로 되돌린다.
시간복잡도 : 평균적으로 빠르지만, 최악의 경우 **O(N)** 이다.

# 주의사항

## 1. 현재 상태를 기준으로 다음 단계로 넘어간다. 

`next_permutation`은 현재 컨테이너의 상태에서 시작해 오름차순으로 순열을 만들기 시작한다.
최종 순열이 내림차순이 되었을 때 까지 실행한다.

- 예: `{3,1,2}`로 실행하면 `(3,1,2)`, `(3,2,1)` 2개가 출력된다.
따라서 전체 순열을 보고 싶은 경우 오름차순으로 정렬된 상태에서 시작해야 한다.

## 2. 중복이 있는 원소들은 중복을 제외하고 순열을 만들어 준다.

`next_permutation`의 강력한 장점은 중복된 원소가 있어도 중복된 순열을 만들지 않는다는 점이다.
- 예: `{1, 1, 2}`로 실행하면 `{1, 1, 2}`, `{1, 2, 1}`, `{2, 1, 1}` 세 가지만 정확히 출력한다.

# 사용 예시

```c++
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
	vector<int> v = { 3,1,2 };
	sort(v.begin(), v.end());

	do {
		for (auto it = v.begin(); it != v.end(); ++it) {
			cout << *it << ' ';
		}
		cout << "\n";
	} while (next_permutation(v.begin(), v.end()));
}
