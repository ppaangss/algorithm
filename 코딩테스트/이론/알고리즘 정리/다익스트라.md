
다익스트라 알고리즘은 하나의 시작 정점에서 다른 모든 정점까지의 최단 경로를 찾는 알고리즘

## 원리

> [!tip] 핵심
> 현재 알고 있는 최단 거리 중 가장 짧은 곳을 먼저 방문하면, 그곳까지의 최단 거리는 변하지 않는다.

1. 출발점 설정
	- 시작 노드의 거리를 0으로, 나머지 모든 노드의 거리를 무한대로 초기화한다.
2. 노드 선택
	- 아직 방문하지 않는 노드 중 현재 최단 거리가 가장 짧은 노드를 선택한다.
3. 거리 갱신
	- 선택한 노드를 거쳐서 인접 노드로 가는 거리가 기존에 알고 있던 거리보다 짧으면 값을 갱신한다.
	- dist[next] = min( dist[next],dist[current] + weight )
4. 반복
	- 모든 노드를 방문할 때 까지 2~3번 과정을 반복한다.

## 특징

- 우선순위 큐
	- 최소 힙 구조인 우선순위 큐를 사용하여 거리가 짧은 순서대로 큐에서 꺼내 처리한다.

- 시간복잡도 (우선순위 큐를 사용할 경우)
	- $O( V log V + E log V )$ 이지만 보통 간선의 수가 많음
	- $O( E log V )$ 
	- V는 정점 수, E는 간선 수
	- 배열을 쓸 경우 $O( V^2 )$

- 음수 가중치 불가
	- 다익스트라는 한 번 방문한 노드의 최단 거리는 확정되었다고 가정하기 때문에 만약 가중치가 음수라면 이후 거리가 줄어들 가능성이 생겨 논리가 깨진다.

- 방향성
	- 방향 그래프, 무방향 그래프 둘 다 사용 가능하다.


## 구현 예시

```c++
#include <iostream>
#include <vector>
#include <queue>

using namespace std;

const int INF = 1e9; // 무한대 값 설정

// (비용, 목적지 노드) 쌍을 저장
typedef pair<int, int> Node;

void dijkstra(int start, int n, vector<Node> adj[]) {
    vector<int> dist(n + 1, INF); // 최단 거리 테이블
    // 최소 힙(Min-priority queue) 생성: {거리, 노드 번호}
    priority_queue<Node, vector<Node>, greater<Node>> pq;

    dist[start] = 0;
    pq.push({0, start});

    while (!pq.empty()) {
        int current_dist = pq.top().first;
        int curr = pq.top().second;
        pq.pop();

        // 현재 꺼낸 거리가 이미 저장된 거리보다 크다면 무시 (이미 처리된 노드)
        if (dist[curr] < current_dist) continue;

        // 인접한 노드 확인
        for (auto& edge : adj[curr]) {
            int next_node = edge.second;
            int next_dist = current_dist + edge.first;

            // 기존 거리보다 짧은 경로를 찾은 경우
            if (next_dist < dist[next_node]) {
                dist[next_node] = next_dist;
                pq.push({next_dist, next_node});
            }
        }
    }

    // 결과 출력
    for (int i = 1; i <= n; i++) {
        if (dist[i] == INF) cout << "INF ";
        else cout << dist[i] << " ";
    }
}

int main() {
    int v = 5, e = 6; // 정점 5개, 간선 6개 예시
    vector<Node> adj[6];

    // adj[시작] = {가중치, 끝}
    adj[1].push_back({2, 2});
    adj[1].push_back({3, 3});
    adj[2].push_back({4, 3});
    adj[2].push_back({5, 4});
    adj[3].push_back({1, 4});
    adj[4].push_back({1, 5});

    dijkstra(1, v, adj);

    return 0;
}
```
