
하위 문제들의 해를 재사용해 중복 계산을 줄이자.!

어떤 경우에 쓰나?
1. 최적 부분 구조
	- 큰 문제의 최적 해가 작은 문제들의 최적 해를 결합해 구할 수 있는 구조
	- ex) 피보나치 F(n) = F(n-1) + F(n-2)
	- 점화식을 세울 수 있으면 좋다.
2. 겹치는 부분 문제
	- 동일한 하위 문제가 여러번 반복되는 경우
	- ex) 피보나치
		- 예를들어 F(4)를 구하는 경우 F(3) + F(2) 이다.
		- 근데 F(3)의 경우 또 F(2) + F(1) 이다.
		- 이때 F(2)가 들어가는 경우가 2번인데 이것을 재귀로 구하면 2번 구하게 된다. 이거를 방지해 미리 F(2)의 값을 저장해 재사용하는 것이 DP 방식이다.
