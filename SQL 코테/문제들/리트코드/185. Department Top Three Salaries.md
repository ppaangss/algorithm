# 개요
### [185. Department Top Three Salaries](https://leetcode.com/problems/department-top-three-salaries/)
### 유형

- CTE
- 윈도우 함수: DENSE_RANK()

### 회고

DENSE_RANK를 정확히 쓸 수 있을 만큼 공부해야한다. DENSE_RANK라는 것을 쓸 준비까지는 했는데 쓰는게 살짝 어려움이 있었다.

그리고 윈도우함수를 WHERE절 아래에서 직접적으로 사용하지는 못한다. 결국엔 CTE를 활용하여 한 번 더 거쳐야 한다.

윈도우함수에서 GROUP BY는 PARITITION BY 이다. !!


# 풀이

DENSE_RANK를 사용하여 랭크를 구한뒤 3등 이내를 추출해 출력해주면 된다.

```sql
WITH RK_INFO AS (
    SELECT 
        name, 
        salary, 
        departmentId, 
        DENSE_RANK() OVER (PARTITION BY departmentId ORDER BY salary DESC) AS 'RK'
    FROM Employee
)
SELECT D.name AS Department , R.name AS Employee, R.Salary 
FROM RK_INFO R
JOIN Department D ON D.id = R.departmentId 
WHERE RK <= 3
;
```
