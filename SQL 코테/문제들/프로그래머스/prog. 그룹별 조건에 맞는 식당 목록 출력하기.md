# 회고

이 문제는 간단히 그냥 동수에 상관없이 가장 먼저 오는 사람 1명에 대한 리뷰를 전부 가져오면 되는 간단한 문제였다.

근데 나는 잘못봐서 동점인 사람 모두의 리뷰를 가져오는 문제인 줄 알았다.

그래도 만약에 여러명을 가져오는 문제였다면 푸는 방식이 2가지이다.

서브쿼리를 오지게 넣거나.
아니면 RANK 윈도우 함수를 쓰거나

# 풀이

정답

```sql
SELECT M.MEMBER_NAME, REVIEW_TEXT, DATE_FORMAT(REVIEW_DATE,'%Y-%m-%d') AS REVIEW_DATE
FROM MEMBER_PROFILE M
JOIN REST_REVIEW R ON M.MEMBER_ID = R.MEMBER_ID
WHERE M.MEMBER_ID = (
    SELECT MEMBER_ID
    FROM REST_REVIEW
    GROUP BY MEMBER_ID
    ORDER BY COUNT(*) DESC
    LIMIT 1
)
ORDER BY REVIEW_DATE ASC, REVIEW_TEXT ASC;
```



여러명이 가능할 경우 

서브쿼리

```sql
SELECT M.MEMBER_NAME, REVIEW_TEXT, DATE_FORMAT(REVIEW_DATE,'%Y-%m-%d') AS REVIEW_DATE
FROM MEMBER_PROFILE M
JOIN REST_REVIEW R ON M.MEMBER_ID = R.MEMBER_ID
WHERE M.MEMBER_ID in (
    SELECT MEMBER_ID
	FROM REST_REVIEW
	GROUP BY MEMBER_ID
	HAVING COUNT(*) = (
	    SELECT COUNT(*) 
		FROM REST_REVIEW 
		GROUP BY MEMBER_ID 
		ORDER BY COUNT(*) DESC 
		LIMIT 1
	)
)
ORDER BY MEMBER_NAME ASC, REVIEW_DATE ASC, REVIEW_TEXT ASC;
```

RANK()

```sql
SELECT M.MEMBER_NAME, REVIEW_TEXT, DATE_FORMAT(REVIEW_DATE,'%Y-%m-%d') AS REVIEW_DATE
FROM MEMBER_PROFILE M
JOIN REST_REVIEW R ON M.MEMBER_ID = R.MEMBER_ID
WHERE M.MEMBER_ID in (
    SELECT MEMBER_ID
    FROM (
        SELECT MEMBER_ID, RANK() OVER (ORDER BY COUNT(*) DESC) as rnk
        FROM REST_REVIEW
        GROUP BY MEMBER_ID
    ) AS Temp
    WHERE rnk = 1
)
ORDER BY MEMBER_NAME ASC, REVIEW_DATE ASC, REVIEW_TEXT ASC;
```
