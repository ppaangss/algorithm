# ROUND

숫자를 특정 소수점 자리에서 반올림할 때 사용한다.

- `ROUND(대상_숫자, 위치)`
- ex) 123.456 을 123으로 반환하기
	- `ROUND(123.456, 0)  ` (소수점 첫째 자리에서 반올림하여 일의 자리까지 표시)
- ex) 123.456 을 123.5로 반환하기 
	- `ROUND(123.456, 1)` (소수점 둘째 자리에서 반올림하여 첫째 자리까지 표시)

### 주의

123.456 을 소수 둘째자리에서 반올림하세요 -> 첫째자리 까지 나타내라는 뜻이다.

즉, `ROUND(123.456, 1)` 을 해야한다.
