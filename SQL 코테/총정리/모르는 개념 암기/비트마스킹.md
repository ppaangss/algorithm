# 비트마스킹

### 비트 연산자 종류

| 연산자        | 기능            | 용도                       |
| ---------- | ------------- | ------------------------ |
| `&` (AND)  | 양쪽 다 1이면 1    | 특정 비트가 켜져 있는지 확인         |
| `\|` (OR)  | 한쪽이라도 1이면 1   | 특정 비트를 추가(ON)            |
| `^` (XOR)  | 서로 다르면 1      | 비트를 반전(Toggle)           |
| `~` (NOT)  | 0은 1로, 1은 0으로 | 특정 비트를 제거(OFF) 할 때 조합 사용 |
| `<<`, `>>` | 비트 밀기         | $2^n$ 값을 계산하거나 자릿수 이동    |

### 비트 연산

비트 연산은 숫자를 값이 아니라 스위치들의 집합으로 보는 것이 핵심이다.

# 예시코드

[조건에 맞는 개발자 찾기](https://school.programmers.co.kr/learn/courses/30/lessons/276034)

```sql
SELECT ID, EMAIL, FIRST_NAME, LAST_NAME
FROM DEVELOPERS
WHERE (SKILL_CODE & (SELECT CODE FROM SKILLCODES WHERE NAME = 'Python'))
OR (SKILL_CODE & (SELECT CODE FROM SKILLCODES WHERE NAME = 'C#'))
ORDER BY ID ASC;
```

### 포함 여부 확인

특정 스위치가 켜져 있는가? 를 확인한다.

#### 예시: 한 유저의 SKILL_CODE 에 Python이 포함되어 있는가?

`(SKILL_CODE & 16) > 0` 라고 표현할 수 있다.

### 여러 개 중 하나라도 포함됐는지 확인

특정 스위치가 단 하나라도 켜져 있는지 확인하는 것이다.

비트연산이므로 모든 경우를 SUM으로 묶어서 & 연산으로 비교할 수 있다.

결과가 0 이상이면 단 하나라도 포함된 것 이다.

#### 예시: 한 유저의 SKILL_CODE에 Back End 기술이 포함되어 있는가?

`SKILL_CODE & (4 + 128) > 0` 라고 표현할 수 있다.

### 여러 개 중 모두 포함됐는지 확인

특정 스위치가 모두 켜져 있는지 확인하는 것이다.

비트연산이므로 모든 경우를 SUM으로 묶어서 & 연산으로 비교할 수 있다.

결과가 SUM으로 묶은 결과와 같아야만 모두 포함된 것 이다.

#### 예시: 한 유저의 SKILL_CODE에 Back End 기술 모두가 포함되어 있는가?

`SKILL_CODE & (4 + 128) = (4 + 128)` 라고 표현할 수 있다.
