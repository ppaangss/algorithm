# YEAR, MONTH, DAY

- ex) DATE 컬럼에서 2021년인 데이터 추출하기
- YEAR 함수 사용하기
	- `WHERE YEAR(DATE) = 2021`
- LIKE 사용하기 (데이터가 맨 앞이 연도일 경우 사용 가능, 데이터 형태에 따라 달라짐)
	- `WHERE DATE LIKE '2021%'
- 범위 지정
	- `WHERE DATE BETWEEN '2021-01-01' AND '2021-12-31'

| 함수         | 추출 대상 | 예시 (결과)                     |
| ---------- | ----- | --------------------------- |
| `YEAR()`   | 연도    | `YEAR('2026-03-27')` → 2026 |
| `MONTH()`  | 월     | `MONTH('2026-03-27')` → 3   |
| `DAY()`    | 일     | `DAY('2026-03-27')` → 27    |
| `HOUR()`   | 시     | `HOUR('16:45:00')` → 16     |
| `MINUTE()` | 분     | `MINUTE('16:45:00')` → 45   |
| `SECOND()` | 초     | `SECOND('16:45:00')` → 0    |
# 날짜 구간 
### 1. BETWEEN AND

BETWEEN 과 AND를 사용해서 날짜 사이를 구할 수 있다.

- 날짜 말고 다른 숫자도 가능하다.
- 이상, 이하를 나타낸다. (초과, 미만 X)

```sql
SELECT *
FROM ANIMAL_INS
WHERE DATETIME BETWEEN '2017-03-01' AND '2017-05-01';

SELECT *
FROM ANIMAL_INS
WHERE DATETIME  NOT BETWEEN '2017-03-01' AND '2017-05-01';
```

### 2. 비교연산자

날짜 데이터에 시간정보가 포함되어 있으면 비교연산자가 더 정확하다.

```sql
SELECT * 
FROM orders 
WHERE order_date >= '2026-03-01' AND order_date < '2026-04-01';
-- 3월 1일 00:00:00부터 4월 1일 00:00:00 '미만'까지 (즉, 3월 전체)

SELECT * FROM orders
WHERE order_date < '2026-03-01' OR order_date > '2026-03-15';
-- 3월 1일 이전이거나, 3월 15일 이후인 데이터
```

# 어떤 연도 (월, 일 포함) 를 구하기

날짜 형태가 정해진 경우 `2022-01-01` + 어떤 연도, 월을 포함한 날짜만을 계산하기를 원하는 경우 구할 수 있는 방법

예를들어 2022년 01월에 구매한 물건을 찾아야할 때, 날짜 형태가 `2022-01-01` 이다.
### like 연산자

그러면 `LIKE '2022-01%'` 이런식으로  구할 수 있다.

### DATE_FORMAT

`DATE_FORMAT(SALES_DATE, '%Y-%m') = '2022-01'` 이런식으로도 구할 수 있다.
