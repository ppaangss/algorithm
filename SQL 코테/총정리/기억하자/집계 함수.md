# 집계함수들

| 함수명       | 역할   | 예시                    |
| ------------- | -------- | ------------------------- |
| `COUNT()` | 행의 개수 세기 | `COUNT(*)` (전체 데이터 개수)    |
| `SUM()`   | 합계 구하기   | `SUM(DAILY_FEE)` (대여료 총합) |
| `AVG()`   | 평균 구하기   | `AVG(DAILY_FEE)` (대여료 평균) |
| `MAX()`   | 최댓값 찾기   | `MAX(PRICE)` (가장 비싼 가격)   |
| `MIN()`   | 최솟값 찾기   | `MIN(PRICE)` (가장 싼 가격)    |
# 주의사항
- `AVG()` 나 `SUM()` 은 NULL 을 제외하고 계산한다. 
	- 만약 NULL을 0으로 치환해서 계산하고 싶다면 `IFNULL()`이나 `COALESCE()`를 함께 사용해야 합니다.
- WHERE vs HAVING
	* `WHERE`: 집계 전에 데이터를 필터링할 때 사용합니다.
	- `HAVING`: 집계 **후**에 그룹화된 결과에 조건을 걸 때 사용합니다.
	- `HAVING AVG(price) > 10000` (평균가가 만 원 이상인 그룹만 출력)
- GROUP BY를 쓰지 않아도 쓸 수 는 있다.

```sql
-- NULL을 제외한 평균 (90점)
SELECT AVG(score) FROM exam;

-- NULL을 0으로 치환한 평균 (60점)
SELECT AVG(IFNULL(score, 0)) FROM exam;
```
