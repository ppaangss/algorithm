# DATE_FORMAT

`DATE_FORMAT` 은 날짜 데이터에 대한 형식을 정할 수 있는 날짜 포맷 함수이다.

`DATE_FORMAT(날짜컬럼, '형식')`

- `DATE_FORMAT(DATE, '%Y-%m-%d')`
	- 2001-05-06


```sql
SELECT DATE_FORMAT(DATETIME,'%Y-%m-%d') AS '날짜'
FROM ...
```

# 자주쓰는 포맷 형식

`%Y-%m-%d` 에서 형태가 대문자와 소문자에 따라 나오는 출력 형식이 달라진다.

- DATE_FORMAT(DATE, "%Y-%m-%d")
	- 2001-05-06
- DATE_FORMAT(DATE , "%y-%m-%d")
	- 01-05-06
 - DATE_FORMAT(DATE , "%Y-%M-%D")
	 - 2001-MAY-6th 

# 그 외 포맷 형식

| 지정자                  | 설명                      | 예시             |
| -------------------- | ----------------------- | -------------- |
| `%Y`                 | 4자리 연도                  | 2026           |
| `%y`                 | 2자리 연도                  | 26             |
| `%M`                 | 월 이름 (Full name)        | March, January |
| `%b`                 | 월 이름 약어 (Abbreviated)   | Mar, Jan       |
| `%m`                 | 월 숫자 (01-12)            | 03, 11         |
| `%c`                 | 월 숫자 (1-12, 0 제외)       | 3, 11          |
| `%d`                 | 일 숫자 (01-31)            | 05, 27         |
| `%e`                 | 일 숫자 (1-31, 0 제외)       | 5, 27          |
| `%D`                 | 서수 표기 포함된 일             | 27th, 1st      |
| `%W`                 | 요일 이름 (Full name)       | Friday, Monday |
| `%a`                 | 요일 이름 약어                | Fri, Mon       |
| `%w`                 | 요일 숫자 (0=일요일, 6=토요일)    | 5 (금요일 기준)     |
| `%H`             | 24시간 형식 시간 (00-23)      | 13, 21         |
| `%h`             | 12시간 형식 시간 (01-12)      | 01, 09         |
| `%i`             | 분 (00-59)               | 05, 45         |
| `%S` 또는 `%s` | 초 (00-59)               | 00, 59         |
| `%p`             | 오전/오후 (AM 또는 PM)        | AM, PM         |
| `%T`             | 24시간 전체 시각 (`hh:mm:ss`) | 14:30:05       |
