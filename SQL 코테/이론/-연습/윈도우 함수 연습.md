# 부서별, 봉급 순위를 추출하자. (동일 봉급은 동일 순위)

윈도우 함수 `rank()` 를 사용하면 된다.

![](../files/Pasted%20image%2020260331131156.png)

```sql
select id,name,dept,salary,
rank() over (partition by dept order by salary desc) as '순위'
from employees;
```

# 부서별 봉급 순위 2위만 추출해보자

먼저 이걸 해보았다.

그래서 간단하게 작성해봤는데 실패했다.

```sql
select id, name, dept, salary
from employees
where 2 in (
select rank() over (partition by dept order by salary desc) as '순위'
from employees
);
```

### 윈도우 함수는 SELECT 절에서만 정의할 수 있다.

윈도우 함수는 `SELECT` 절에서만 정의할 수 있다. `WHERE`, `GROUP BY`, `HAVING` 절에서는 윈도우 함수를 직접 사용할 수 없다.

윈도우 함수는 `WHERE` 절로 데이터를 다 걸러낸 다음에 계산된다. 즉, `WHERE` 절 입장에서는 아직 '순위' 라는 데이터가 존재하지 않아서 비교를 할 수 없다.

방법은 2가지로 가능하다.
1. 서브쿼리
2. CTE 사용 (WITH)

두 가지 모두 방식은 비슷하다. 순위까지 만든 테이블을 from에서 이용하는 것이다.

![](../files/Pasted%20image%2020260331132039.png)
### 서브쿼리를 이용한 해결

```sql
SELECT id, name, dept, salary, rank_val
FROM (
    SELECT id, name, dept, salary,
           RANK() OVER (PARTITION BY dept ORDER BY salary DESC) as rank_val
    FROM employees
) as temp_table
WHERE rank_val = 2;
```

### CTE 사용

```sql
WITH RankedEmployees AS (
    SELECT id, name, dept, salary, hire_date,
           RANK() OVER (PARTITION BY dept ORDER BY salary DESC) as rnk
    FROM employees
)
SELECT id, name, dept, salary, hire_date, rnk
FROM RankedEmployees
WHERE rnk = 2;
```
