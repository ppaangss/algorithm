
## 1. 문자열 결합 및 길이 확인

데이터를 보기 좋게 합치거나, 데이터의 유효성을 체크할 때 사용합니다.

- **`CONCAT(s1, s2, ...)`**: 여러 문자열이나 컬럼 값을 하나로 합칩니다.
    
    - `SELECT CONCAT(first_name, ' ', last_name) AS full_name FROM users;`
        
- **`LENGTH(s)` (또는 `LEN`)**: 문자열의 길이를 반환합니다.
    
    - `SELECT LENGTH('Hello'); -- 결과: 5`
        
- **`CONCAT_WS(separator, s1, s2, ...)`**: 설정한 구분자(WS: With Separator)를 사이에 넣고 합칩니다.
    
    - `SELECT CONCAT_WS('-', '2026', '03', '31'); -- 결과: 2026-03-31`
        

---

## 2. 문자열 자르기 및 추출

긴 텍스트에서 필요한 부분만 골라낼 때 필수적입니다.

- **`SUBSTR(s, start, [length])` (또는 `SUBSTRING`)**: 특정 위치부터 지정한 길이만큼 자릅니다. **(대부분의 SQL에서 인덱스는 1부터 시작합니다)**
    
    - `SELECT SUBSTR('20260331', 1, 4); -- 결과: 2026`
        
- **`LEFT(s, n)` / `RIGHT(s, n)`**: 왼쪽이나 오른쪽에서부터 `n`글자를 가져옵니다.
    
    - `SELECT LEFT('SQL_Study', 3); -- 결과: SQL`
        
- **`TRIM(s)`**: 문자열 양쪽의 공백을 제거합니다. (`LTRIM`, `RTRIM`은 각각 왼쪽, 오른쪽만 제거)
    

---

## 3. 치환 및 대소문자 변환

데이터의 형식을 통일하거나 특정 단어를 바꿀 때 유용합니다.

- **`REPLACE(s, 'old', 'new')`**: 특정 문자를 찾아 다른 문자로 바꿉니다.
    
    - `SELECT REPLACE('010-1234-5678', '-', ''); -- 결과: 01012345678`
        
- **`UPPER(s)` / `LOWER(s)`**: 모두 대문자 또는 소문자로 변환합니다. 검색 조건의 대소문자를 무시하고 싶을 때 자주 씁니다.
    
    - `SELECT * FROM users WHERE LOWER(email) = 'test@gmail.com';`
        

---

## 4. 위치 찾기

특정 문자가 문자열 내에 어디에 있는지 확인할 때 사용합니다.

- **`INSTR(s, target)` (또는 `CHARINDEX`, `POSITION`)**: 찾는 문자가 처음 나타나는 위치(인덱스)를 반환합니다.
    
    - `SELECT INSTR('apple@gmail.com', '@'); -- 결과: 6`

### 💡 실무 꿀팁: 함수 중첩 사용

함수는 여러 개를 겹쳐서 쓸 수 있습니다. 예를 들어, 이메일 주소에서 아이디만 추출하고 싶다면 이렇게 할 수 있죠.

SQL

```
-- 이메일에서 @ 앞부분만 잘라내기
SELECT SUBSTR(email, 1, INSTR(email, '@') - 1) AS user_id
FROM users;
```
